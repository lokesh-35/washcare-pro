# WashCare Pro

A responsive car service landing page built with Vite, React, and TypeScript.

## Local development

1. Install dependencies:
   `npm install`
2. Start the dev server:
   `npm run dev`
3. Open the URL shown in the terminal.

## Build

Run:

`npm run build`

The static site will be generated into the `dist/` folder.

## Deployment

This project includes `vercel.json` for Vercel static deployment.

> Ensure the GitHub repository is connected to Vercel and deploy the project from the `main` branch.
